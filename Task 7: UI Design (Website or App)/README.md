# 💻 Task 07 – UI Design (Website)

## 📌 Objective

Design a modern, user-friendly website interface with a clean layout and intuitive user experience.

---

## 📝 Project Description

This project presents a **website UI design** for an **IT Learning Platform** created using **Figma**. The interface focuses on providing an engaging learning experience through a clean layout, clear navigation, and visually appealing components.

The homepage is designed with modern UI principles to help users easily explore courses, learn about the platform, and start their learning journey.
