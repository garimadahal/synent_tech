## Project Overview

This project is a creative social media post designed for Instagram using Figma. The objective was to create an engaging and visually appealing promotional post with a clear message, attractive layout, and effective call-to-action.

**Theme:**
> "Design That Speaks Louder Than Words."

---

## Objective

- Design an engaging Instagram social media post.
- Create a clean and modern layout.
- Apply proper typography, color harmony, and visual hierarchy.
- Include a Call-to-Action (CTA).

---

## Tools Used

- Figma

---

## Design Elements

- Synent Technologies Logo
- Creative Headline
- Supporting Description
- Smartphone Mockup
- Call-to-Action Button
- Website URL
- Social Media Icons

---

## Color Palette

- Purple
- Cream
- Black
- White

These colors were selected to represent creativity, professionalism, and simplicity.

---

## Typography

The design uses bold typography for the headline and readable fonts for the body text to improve visual hierarchy and readability.

---

## Design Process

1. Planned the layout.
2. Selected the color palette.
3. Added branding elements.
4. Created the headline and supporting content.
5. Inserted a smartphone mockup.
6. Designed the CTA button.
7. Arranged the final composition and spacing.

---

## Features

- Clean and modern design
- Balanced layout
- Professional color scheme
- Clear Call-to-Action
- Instagram-friendly square format
- Brand-focused visual presentation

---

## Output

- Instagram Social Media Post (1080 × 1080 px)
