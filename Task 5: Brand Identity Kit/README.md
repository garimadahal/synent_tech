# 🎨 Task 05 – Brand Identity Kit

## 📌 Objective

Create a complete brand identity kit that defines the visual identity of the brand through consistent logo usage, color palette, typography, and brand guidelines.

---

## 📝 Project Description

This project presents a comprehensive **Brand Identity Kit** for **Synent Technologies**, designed using **Figma**. The purpose of this kit is to establish a consistent visual language that can be applied across digital and print platforms.

The brand guideline document includes the company logo, color palette, typography, logo usage rules, and brand mockups to ensure consistency in future branding materials.

---

## ✨ Features

- Professional Brand Identity Guide
- Logo Variations
- Brand Color Palette
- Typography Guidelines
- Logo Clear Space
- Logo Do's & Don'ts
