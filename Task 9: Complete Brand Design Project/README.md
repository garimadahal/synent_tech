# 🎨 Task 09 – Complete Brand Design Project

## 📌 Objective

Create a complete brand design project by developing a professional and consistent visual identity for a brand. The project includes essential branding assets that can be used across both digital and print platforms.

---

## 📝 Project Description

This project presents a complete **Brand Design Project** for **Synent Technologies & EduLearn**, designed using **Figma**. The objective was to build a cohesive brand identity by designing the company's logo, business card, social media post, and promotional banner while maintaining consistency in colors, typography, and overall visual style.

The project demonstrates the application of branding principles to create a recognizable and professional identity.

---

- Business Card (Front & Back)
