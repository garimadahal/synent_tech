# 📢 Task 08 – Marketing Creatives Pack

## 📌 Objective

Design a complete marketing creatives pack that promotes a brand through visually engaging and consistent marketing materials.

---

## 📝 Project Description

This project showcases a **Marketing Creatives Pack** created for **Synent Technologies** using **Figma**. The design package includes a promotional social media post, a marketing banner, and a promotional poster, all following a unified visual identity.

The goal of this project is to create professional marketing assets that effectively communicate the brand message while maintaining consistency across different platforms.
