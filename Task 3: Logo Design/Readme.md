# 🎨 Task 03 – Logo Design

## 📌 Objective

Design a unique, minimal, and memorable logo that represents the identity of the brand while maintaining a modern and professional appearance.

---

## 📝 Project Description

This project presents the official logo design for **Synent Technologies**. The logo was created using **Figma** with a minimalist design approach to reflect innovation, professionalism, and technology. The design focuses on simplicity, scalability, and versatility, making it suitable for both digital and print applications.

---

## ✨ Features

- Modern and minimal logo design
- Unique brand identity
- Scalable vector design
- Works on both light and dark backgrounds
- Professional typography
- Multiple logo variations
