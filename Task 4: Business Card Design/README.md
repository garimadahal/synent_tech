# 💼 Task 04 – Business Card Design

## 📌 Objective

Design a professional business card that represents the brand identity while providing clear and organized contact information.

---

## 📝 Project Description

This project showcases a modern **front and back business card design** created for **Synent Technologies** using **Figma**. The design follows the company's branding by incorporating the official logo, color palette, typography, and a clean layout to create a professional and memorable first impression.

---

## ✨ Features

- Professional front and back design
- Clean and minimal layout
- Consistent brand identity
- Company logo integration
- Readable typography
- Balanced spacing and alignment
- Print-ready design
